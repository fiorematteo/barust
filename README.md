# Barust

A X11 status bar.

Barust uses EWMH to get informations from the WM.

## Features

Barust uses a widgets model inspired by [qtile](http://www.qtile.org/) and [cnx](https://github.com/mjkillough/cnx).

## Widgets

* battery
* clock
* cpu
* memory
* network
* spacer
* svg
* systray (WIP)
* temperature
* text
* workspaces
